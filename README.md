# Calculadora de Redes

Projeto desenvolvido no 2º semestre do curso de Engenharia de Software, para a disciplina de Rede de Computadores 


 ## ⚙ Funcionalidades

Ao inserir endereço de IP classe A, B ou C, o programa exibe:
- IP base
- Classe
- Máscara Padrão
- Subnet 
- Hosts Adicionados
- Broadcast


## 🛠 Tecnologias 

- Windows Form C#
